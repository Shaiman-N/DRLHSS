## üöÄ Deployment Options

### Windows

**Installer**: MSI (Windows Installer)
- Size: ~50 MB
- Requirements: Windows 10/11
- Installation: Double-click MSI
- Service: Windows Service
- Shortcuts: Start Menu, Desktop

**Installation**:
```powershell
# Download installer
Invoke-WebRequest -Uri "https://downloads.direwolf.ai/direwolf-1.0.0-windows-x86_64.msi" -OutFile "direwolf.msi"

# Install
msiexec /i direwolf.msi /qn

# Run
direwolf
```

### Linux

**Packages**: DEB, RPM, AppImage
- Size: ~45-55 MB
- Requirements: Ubuntu 20.04+, Fedora 35+, or any modern Linux
- Installation: Package manager or AppImage
- Service: systemd
- Integration: Desktop file

**Installation**:
```bash
# Debian/Ubuntu (DEB)
wget https://downloads.direwolf.ai/direwolf-1.0.0-linux-x86_64.deb
sudo dpkg -i direwolf-1.0.0-linux-x86_64.deb

# Red Hat/Fedora (RPM)
wget https://downloads.direwolf.ai/direwolf-1.0.0-linux-x86_64.rpm
sudo rpm -i direwolf-1.0.0-linux-x86_64.rpm

# Universal (AppImage)
wget https://downloads.direwolf.ai/direwolf-1.0.0-linux-x86_64.AppImage
chmod +x direwolf-1.0.0-linux-x86_64.AppImage
./direwolf-1.0.0-linux-x86_64.AppImage

# Run
direwolf
```

### macOS

**Packages**: DMG, PKG
- Size: ~50 MB
- Requirements: macOS 11.0+
- Installation: Drag-to-install or PKG
- Service: launchd
- Integration: Applications folder

**Installation**:
```bash
# Download DMG
curl -O https://downloads.direwolf.ai/direwolf-1.0.0-macos-x86_64.dmg

# Mount and install
hdiutil attach direwolf-1.0.0-macos-x86_64.dmg
cp -R /Volumes/DIREWOLF/DIREWOLF.app /Applications/
hdiutil detach /Volumes/DIREWOLF

# Run
open /Applications/DIREWOLF.app
```
